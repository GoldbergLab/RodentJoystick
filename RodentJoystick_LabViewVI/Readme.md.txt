To start the labview code open the Project File: 

./Rodent Joystick VI/ControlPanel_project

FPGA_main is the VI that runs on the FPGA in th esb-RIO
RT_main is the VI that runs on the Real time processor on the sb-RIO
PC_main is the VI that runs on the local PC and stores data

Designed for Labview2013, also works on Labview2015 with some minor tweaks.
